# Changelog

All notable changes to this project will be documented in this file.

## [2.4.0] - 2026-01-20
### Added
- **Orin NX Support**: Native CUDA bindings for Jetson L4T 35.4.1.
- **Virtual Agent**: Added `aeva.virtual` for headless simulation.
- **Safety**: Implemented ISO-10218 compliant "Red Button" E-Stop loop.

### Changed
- Rebranded from AEVA to **ThemisAevra**.
- Updated `pyproject.toml` dependencies.

## [2.3.0] - 2025-11-15
### Added
- Initial release of `Spatial-Net` inference engine.
- gRPC telemetry streaming.
